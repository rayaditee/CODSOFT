# Task 4 - Simple Recommendation System

This project implements a simple content-based recommendation system using Python.

## Objective
Recommend movies based on user preferences using content-based filtering.

## Technique Used
Content-Based Filtering

## How It Works
- Movies are stored with their features (genres/tags)
- User preferences are matched with movie features
- Movies are ranked based on matching score

## Technologies Used
- Python

## Output
The system recommends movies based on how closely they match the user's interests.
